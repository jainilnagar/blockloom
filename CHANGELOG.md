# Changelog

## 1.0.0

- Initial release
- Icon block — FA Free inline SVG, hover animations, accessibility label, link rel control
- Icon List — vertical/horizontal layout, responsive flex wrapping, icon position, per-item icon picker
- Icon List Item — server-side rendered via render.php, PHP SVG helper with static caching
- Info Box — three layouts, content alignment, title tag selector, card hover states, linked padding
- FAQs — CSS grid animation, context-driven icon in editor, icon position, FAQPage JSON-LD schema
- Team Member — seven social networks with FA Brand icons, Person JSON-LD schema, image lazy loading
- Rating — stars (full/half) and bar styles, AggregateRating JSON-LD schema, scroll animation
- Progress Bar — striped fill with CSS animation, scroll animation, custom label, label position
- Shared ColorControl with theme palette and Normal/Hover tabs
- Shared PaddingControl with linked/unlinked sides
- IconPicker with searchable FA Free grid, category filter, and load more pagination
- StarRating component with linearGradient half-star support
- PHP icon helper with static request caching
- Conditional asset loading via has_block()
- WPCS compliant PHP, PHPStan level 5, cross-platform PHP lint script
- .gitattributes with export-ignore for dev files and LF line ending enforcement
